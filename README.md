# Site romântico para Laryssa

Instalação rápida (XAMPP no Windows):

1. Coloque a pasta `projeto` em `C:\xampp\htdocs\programadorweb\` (já presumido).
2. Inicie o Apache via XAMPP.
3. Abra no navegador: `http://localhost/programadorweb/projeto/`

Como colocar a foto:

- Você pode arrastar/selecionar uma foto usando o botão **Enviar foto** na página — o arquivo será salvo em `images/larissa.jpg`.
- Ou copie manualmente sua imagem para `projeto/images/larissa.jpg`.

Controles disponíveis na página:

- **Enviar foto**: abre seletor; envia via AJAX e atualiza a imagem sem recarregar.
- **Seleção de fonte**: altere a tipografia do título.
- **Trocar tema**: alterna entre paletas românticas (rose/teal/sunset).
- **Baixar cartão (PDF)**: abre o diálogo de impressão; escolha "Salvar como PDF" no sistema.

Se quiser que eu coloque exatamente a foto que você enviou aqui, posso também copiar manualmente o arquivo para `images/larissa.jpg` se você autorizar o upload no workspace; caso contrário, use o botão "Enviar foto" na página.
